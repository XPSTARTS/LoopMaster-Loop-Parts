# 🔄 LoopMaster - YouTube A-B Looper

Loop specific parts of YouTube videos with one click!

[![Firefox Add-on](https://img.shields.io/badge/Firefox-Add--on-ff6816?style=for-the-badge&logo=firefox)](https://addons.mozilla.org/addon/loopmaster-youtube-a-b-looper/)

---

## 🎯 What it does

LoopMaster lets you loop any specific part of a YouTube video.

**Perfect for:**
- 🎵 Music practice (loop that guitar riff!)
- 📚 Language learning (repeat phrases)
- 💪 Motivational quotes (drill them in)
- 🎮 Gaming tutorials (replay moves)

---

## 🚀 Installation

### Firefox (Recommended)
[Download from Firefox Add-ons](https://addons.mozilla.org/addon/loopmaster-youtube-a-b-looper/)

### Chrome (Manual)
1. Clone this repo
2. Run `npm install` then `npm run build`
3. Go to `chrome://extensions/`
4. Turn on **Developer mode**
5. Click **"Load unpacked"**
6. Select this folder

**Quick Install:** Double-click `install-chrome.bat`

---

## 🎮 How to Use

1. Play any YouTube video
2. Click **A** at your start point
3. Click **B** at your end point
4. Click **▶** to loop forever!
5. Click **▶** again to stop
6. Click **✕** to clear points

That's it! 🎉
