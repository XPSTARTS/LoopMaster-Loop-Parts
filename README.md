# LoopMaster - YouTube A-B Looper

Loop specific parts of YouTube videos with one click!

## Features
- Set Start (A) and End (B) points
- Keyboard shortcuts: Ctrl+Shift+1,2,3,4
- Visual progress bar highlight
- Works on all YouTube videos

## How to Use
1. Play any YouTube video
2. Click 'A' at your start point
3. Click 'B' at your end point
4. Click '▶' to loop forever!
